Мы рады представить важное обновление, которое сэкономит массу времени нашим продавцам: теперь вы можете активировать виджет виртуальной примерки Genlook сразу для целых коллекций товаров.

Раньше Genlook приходилось включать вручную для каждого товара по отдельности. И хотя такой подход давал полный контроль над настройками, для магазинов с большим каталогом процесс занимал слишком много времени. Теперь всё стало намного проще.

## Как это работает

1.  **Зайдите в панель управления Genlook:** Перейдите в раздел "Product Settings" (Настройки товаров).
2.  **Выберите "Collections":** Там вы увидите новую опцию для управления коллекциями.
3.  **Отметьте нужные коллекции:** Просто выберите группы товаров, для которых хотите включить Genlook.
4.  **Сохраните изменения:** Виджет Genlook автоматически появится на страницах всех товаров, входящих в выбранные коллекции.

Эта функция позволяет гораздо быстрее внедрить виртуальную примерочную в масштабах всего магазина. А значит, вы сможете быстрее начать увеличивать конверсию и снижать процент возвратов.
